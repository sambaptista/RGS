Présentation.

Les dépêches du réseau Jeux-Stratégie sont accessibles via un flux RSS. Pour ceux qui ne connaissent pas le système sachez simplement qu'il permet de savoir si une nouvelle dépêche a été postée sans devoir venir sur le site, ce qui est très pratique pour être au courant des dernières actualités le plus rapidement possible.

Ce système fonctionne sur les navigateurs les plus connus tel que Firefox, Internet Explorer ou encore Opéra. Pour une explication encore plus précise vous pouvez consulter la page Wiki des flux RSS.
